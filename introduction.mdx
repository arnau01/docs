# Welcome to Throxy

Throxy is a powerful B2B lead generation platform that helps you find and connect with the right prospects at scale. Our platform combines advanced targeting capabilities with automated lead finding to streamline your B2B outreach process.

## Key Features

- **Targeted Lead Finding**: Find leads that match your specific personas and criteria
- **Structured Results**: Get your leads in organized, ready-to-use formats

## Getting Started

To start using the Throxy API, you'll need to [speak with Arnau](https://cal.com/arnau-ayerbe/15min).

Once you have these, you can start making API requests to find and manage your leads.

## Authentication

All API endpoints require authentication using an API key. Include your API key in the request headers:

```bash
x-api-key: YOUR_API_KEY
```
