# Get Job Details

Retrieve detailed information about a specific lead finding job, including its configuration, status, and results if completed.

## API Reference

```bash
GET https://app.throxy.com/api/lead-finding/get_job_details
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your API key for authentication
</ParamField>

## Parameters

<ParamField query="job_id" type="string" required>
  The unique identifier of the job to retrieve
</ParamField>

## Response

```json
{
  "job_id": "abc123def456",
  "name": "Tech Leaders Campaign",
  "personas": [
    "CTO at software companies",
    "VP of Engineering",
    "Technical Director"
  ],
  "selected_fields": ["first_name", "last_name", "email", "title"],
  "domains": ["company1.com", "company2.com"],
  "created_at": "2024-01-20T10:00:00.000Z",
  "status": "completed",
  "complete_leads_sheet": "https://sheets.google.com/..."
}
```

### Response Properties

<ResponseField name="job_id" type="string">
  Unique identifier for the job
</ResponseField>

<ResponseField name="name" type="string">
  Name of the lead finding job
</ResponseField>

<ResponseField name="personas" type="array">
  Array of persona strings in priority order (index 0 is highest priority)
</ResponseField>

<ResponseField name="selected_fields" type="array">
  Array of fields selected for lead data collection. Can include:
  - first_name
  - last_name
  - title
  - email
  - country
  - salutation
  - email_or_linkedin_url
  - linkedin_url
  - gender
  - preferred_language
</ResponseField>

<ResponseField name="domains" type="array">
  Array of target domains used for lead finding
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp of when the job was created
</ResponseField>

<ResponseField name="status" type="string">
  Current status of the job. Possible values:
  - ongoing
  - completed
  - failed
</ResponseField>

<ResponseField name="complete_leads_sheet" type="string">
  URL to access the completed leads sheet. Only available when status is "completed"
</ResponseField>

## Status Codes

<ResponseField name="200: Success" type="integer">
  Successfully retrieved job details
</ResponseField>

<ResponseField name="400: Bad Request" type="integer">
  Missing or invalid job_id parameter
</ResponseField>

<ResponseField name="401: Unauthorized" type="integer">
  Invalid or missing API key
</ResponseField>

<ResponseField name="404: Not Found" type="integer">
  Job not found
</ResponseField>

<ResponseField name="500: Internal Server Error" type="integer">
  Server error while fetching job details
</ResponseField>

## Example Request

```bash
curl --request GET \
  --url 'https://app.throxy.com/api/lead-finding/get_job_details?job_id=abc123def456' \
  --header 'x-api-key: YOUR_API_KEY'
```
