# List All Jobs

Retrieve a list of all lead finding jobs associated with your account. Jobs are returned in descending order by creation date (newest first).

## API Reference

```bash
GET https://app.throxy.com/api/lead-finding/get_all_jobs
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your API key for authentication
</ParamField>

## Response

<ResponseField name="jobs" type="array">
  Array of job objects containing basic job information
</ResponseField>

```json
{
  "jobs": [
    {
      "job_id": "abc123def456",
      "name": "Tech Leaders Campaign",
      "status": "completed",
      "created_at": "2024-01-20T10:00:00.000Z",
      "complete_leads_sheet": "https://sheets.google.com/..."
    }
  ]
}
```

### Job Object Properties

<ResponseField name="job_id" type="string">
  Unique identifier for the job
</ResponseField>

<ResponseField name="name" type="string">
  Name of the lead finding job
</ResponseField>

<ResponseField name="status" type="string">
  Current status of the job. Possible values:
  - ongoing
  - completed
  - failed
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp of when the job was created
</ResponseField>

<ResponseField name="complete_leads_sheet" type="string">
  URL to access the completed leads sheet. Only available when status is "completed"
</ResponseField>

## Status Codes

<ResponseField name="200: Success" type="integer">
  Successfully retrieved list of jobs
</ResponseField>

<ResponseField name="401: Unauthorized" type="integer">
  Invalid or missing API key
</ResponseField>

<ResponseField name="500: Internal Server Error" type="integer">
  Server error while fetching jobs
</ResponseField>

## Example Request

```bash
curl --request GET \
  --url https://app.throxy.com/api/lead-finding/get_all_jobs \
  --header 'x-api-key: YOUR_API_KEY'
```
