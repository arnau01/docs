# Get Job Details

## API Reference

Returns detailed information about a specific lead finding job.

```bash
GET /api/lead-finding/get_job_details?job_id={job_id}
```

## Parameters

<ParamField query="job_id" type="string" required>
  The unique identifier of the job to retrieve
</ParamField>

## Response

Returns a detailed job object with the following properties:

```json
{
  "job_id": "abc123",
  "name": "Q4 Engineering Leads",
  "personas": ["Software Engineer", "Developer"],
  "selected_fields": ["first_name", "last_name", "email"],
  "domains": ["example.com", "company.com"],
  "created_at": "2023-11-25T12:00:00Z",
  "status": "ongoing",
  "complete_leads_sheet": null
}
```

### Properties

<ResponseField name="job_id" type="string" required>
  Unique identifier for the job
</ResponseField>

<ResponseField name="name" type="string" required>
  Name of the job. Defaults to "Job {first 6 chars of job_id}" if not specified
</ResponseField>

<ResponseField name="personas" type="array of strings" required>
  Array of persona configurations used for lead finding
</ResponseField>

<ResponseField name="selected_fields" type="array of strings" required>
  Array of fields selected for lead data collection
</ResponseField>

<ResponseField name="domains" type="array of strings" required>
  Array of target domains for lead finding
</ResponseField>

<ResponseField name="created_at" type="string" required>
  ISO 8601 formatted timestamp of when the job was created
</ResponseField>

<ResponseField name="status" type="string" required>
  Current status of the job (e.g., "ongoing", "completed")
</ResponseField>

<ResponseField name="complete_leads_sheet" type="string or null">
  URL to the completed leads sheet, if available
</ResponseField>

## Status Codes

<ResponseField name="200: OK" type="Success">
  Successfully retrieved job details
</ResponseField>

<ResponseField name="400: Bad Request" type="Error">
  Missing or invalid job_id parameter
</ResponseField>

<ResponseField name="401: Unauthorized" type="Error">
  Invalid or missing API key
</ResponseField>

<ResponseField name="404: Not Found" type="Error">
  Job not found
</ResponseField>

<ResponseField name="500: Internal Server Error" type="Error">
  Server error while fetching job details
</ResponseField>
