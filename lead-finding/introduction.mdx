# Lead Finding API

The Lead Finding API allows you to programmatically manage lead generation jobs, retrieve job statuses, and access lead data. This API provides endpoints for scheduling new jobs, monitoring their progress, and fetching completed results.

## Authentication

All API endpoints require authentication using an API key. Include your API key in the request headers:

```bash
Authorization: Bearer YOUR_API_KEY
```

## Base URL

```bash
https://your-domain.com/api/lead-finding
```

## Available Endpoints

- `GET /get_all_jobs` - Retrieve all lead finding jobs for a user
- `GET /get_job_details` - Get detailed information about a specific job
- `POST /schedule_job` - Schedule a new lead finding job
